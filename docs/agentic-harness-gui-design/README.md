# Agentic Harness GUI Design Documentation

This directory contains the design materials that produced the Agentic
Harness Linux GUI (shipped in v0.7.0).

## Documents

| File | Purpose |
|---|---|
| `codex-app-design-workflow-plan.md` | Codex-app-driven design workflow |
| `codex-app-gui-design-goal.md` | Goal spec for the Codex design pass |
| `linux-gui-agentic-harness-plan.md` | Linux/Pop-OS GUI product plan |
| `codex-app-gui-design-result.md` | Final design result from the Codex app |
| `gui-implementation-plan.md` | Plan to convert the design into a working app |
| `codex-handoff-2026-07-10.md` | Codex-to-Codex handoff for implementation |

## Source-of-truth note

These documents are historical design artifacts. The current GUI lives at
`agentic_harness/gui/` and is reachable via `agentic-harness gui`. If a
design change is needed, start by editing the relevant `*.md` here and then
propose the change against the current GUI implementation.
