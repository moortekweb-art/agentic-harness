                    Looking for Ollama, LM Studio, vLLM, and llama.cpp…
                    New to local AI? Start with LM Studio.
                    1. Install LM Studio.
                    2. In Discover, download a chat model that supports tool use or function calling.
                    3. Load the model, then open Developer and switch on Start server.
                    4. Return here and choose Find local AI. Agentic Harness will test the model before saving.
                    Other local servers

                    Ollama, vLLM, and llama.cpp also work through their OpenAI-compatible servers. Use Manual connection when they run on a different port or another computer on your private network.
